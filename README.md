# Flowchart-Laptop-Diagnostics
Este repositorio contiene un Sistema Experto desarrollado en CLIPS diseñado para el diagnóstico de fallas de encendido en computadoras portátiles. El sistema modela el flujo de decisiones lógico para identificar si el problema radica en el suministro de energía (AC/DC), bloqueos lógicos del sistema o fallas críticas de hardware en la placa base.


El motor de inferencia se basa en el reconocido diagrama de flujo de Morris Rosenthal (https://www.ifitjams.com/lappower.htm). Sin embargo, el conocimiento original de 2011 fue actualizado y extendido para abordar las tecnologías de hardware de 2026.
